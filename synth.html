Synthesizer

Using a Propeller 21910 activity board

I created a basic synthesizer as an experiment. By using some wires, resistors and buttons I was able to to turn the breadboard in a keyboard of sorts, and by writing some code in C I was able to assign tones to those buttons.

The Hardware

  • One (1) Propeller 21910 activity board
  • Four (4) buttons
  • Eight (8) Resistors

The Code

The different pitches are generated by rapidly pulsing the pin connected to the audio outpoot of the Propeller Activity Board on and off. This results in a tone. The faster the pin refreshes, the higher the pitch, and vice versa. Here are some examples of code that I used to generate the pitches.

I first set i = input(pin) where pin is whatever pin I have my button connected to. Then I create a while loop and within it an if statement that only runs if the button is pressed. That if statement alternates pin 26 (the audio output pin) on and off to create the pitches for my synthesizer.

Here's an example of code that produces a tone. the 2400 represents that there are approximates 2400 microseconds between each pulse. This is what determines the pitch of the note the while loop creates.


while(1) {

if (i == 1) {
   high(26);
   usleep(2400);
   low(26);
   usleep(2400);
    }
}

In addition to pure tones, different sounds can be coaxed from the board by using different methods in code. By using a for loop in a certain way, a pitch can be generated that starts high and and deepens, not unlike an alarm. Here's how this is accomplished with code:

        
  for(z=0; z < 2000; z++) {
  i = input(5)*(1000+z);
  high(26);
  usleep(i);
  low(26);
  usleep(i);
    }
